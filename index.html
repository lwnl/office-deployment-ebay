Steps to Install and Activate Microsoft Office 2024 Professional Plus

Video demonstration of the installation and activation process

1. Download and Install the Office Deployment Tool

  1. Download link: https://www.microsoft.com/en-in/download/details.aspx?id=49117
  2. Right-click on officedeploymenttool and choose “Run as administrator”. Click Continue, and specify the extraction path — for example, extract it to the Downloads directory.
  3. Then, in the specified path, you will see two extracted files: configuration-Office365-x64.xml and setup.exe.
  4. Rename configuration-Office365-x64.xml to configuration.xml.

2. Start Office Deployment

  1. Right-click on configuration.xml and open it with Notepad.
  2. Since this file contains Microsoft’s default configuration code for downloading Office 365 — which is not the version we need — we will replace it with the correct configuration for Office Pro Plus 2024 Volume. Please replace the content of the file with the following code then save and close it.

    The code to replace with:

    <Configuration>
  <Add OfficeClientEdition="64" Channel="PerpetualVL2024">
    <Product ID="ProPlus2024Volume">
      <Language ID="en-us" />
    </Product>
  </Add>
  <Display Level="Full" AcceptEULA="TRUE" />
</Configuration>

    The default installation language here is English. If you need another language, please add it or replace the default language. You can ask ChatGPT or other AI tools to help you modify the code to change the language.

  3. Right-click in the folder where the setup.exe file is located and select “Open in Terminal.”
  4. Copy the file path — make sure not to include the trailing “>”. Paste it somewhere to save.
  5. In the Windows search bar, type “cmd”, then in the list of results, right-click on “Command Prompt” and select “Run as administrator”.
  6. Type cd /d C:\Users\leo_n\Downloads and press Enter. Here, C:\Users\leo_n\Downloads is the file path you saved earlier.You need to replace it with the file path on your own computer.
  7. Type setup.exe /configure configuration.xml and press Enter.
  8. Microsoft Office will start the download process, which may take ten minutes or longer depending on your internet speed.
  9. After installation is complete, a notification will appear in the bottom-right corner.

4. Activation

  1. Open any Office app (Word, Excel, etc.).
  2. Accept the license agreement
  3. Go to Account → Change Product Key.
  4. Enter your product key and click “Activate Office”.
  5. Close the app and reopen it. When prompted, Choose Phone Activation.
  6. Send us the Installation ID (Step 2 on screen). A screenshot is preferred for accuracy.
  7. Enter the Confirmation ID we provide to complete the activation of your product.

If you do not receive the Confirmation ID right away, please don’t worry. Restarting your computer or the Office program will not affect its validity — the Confirmation ID will still work.

ℹ️ Additional Notes

  • Double-check the Installation ID if activation fails.
  • Your Installation ID is unique and linked to your Confirmation ID.
  • Confirmation ID works even after a restart.
  • All Office apps activate together.
  • Contact us anytime for help.
